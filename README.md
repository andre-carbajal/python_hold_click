## Instrucciones

 1. Install mouse with:
  ```  pip3 install mouse```
 2. Install keyboard with:
 ```  pip3 install keyboard```
 3. Execute coord.py with:
 ```python coord.py```
 4. Copy the x and y position
 5. In main.py, line 7 replace x and y with your coordinates:
 ```
 mouse.move(x, y, duration=  0.1)
```
